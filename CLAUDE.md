# Copywriting

Sempre escreva focando em benefícios.

Nunca comece explicando funcionalidades.

Primeiro explique o benefício.

Depois explique como ele acontece.

Exemplo:

❌ Monitoramos tarifas diariamente.

✅ Você evita perder receita porque monitoramos diariamente o mercado.

Outro exemplo:

❌ Revenue Management.

✅ Mais receita utilizando os mesmos quartos.

Sempre responda mentalmente:

"O que o cliente ganha com isso?"

Antes de finalizar qualquer texto pergunte:

Um proprietário de pousada entenderia isso imediatamente?

Se a resposta for não, simplifique.

Menos texto é melhor.

Mais clareza é melhor.

# Como entregar alterações

Escolha sempre a forma de entrega mais adequada.

Se apenas uma pequena parte foi alterada, entregue somente o trecho modificado indicando exatamente onde ele deve ser inserido.

Se uma alteração estrutural afetar diversos arquivos, apresente todos os arquivos envolvidos.

Explique sempre:

- o que mudou;
- por que mudou;
- quais benefícios trouxe;
- quais arquivos foram alterados.

Nunca envie novamente arquivos completos quando apenas poucas linhas foram modificadas.
